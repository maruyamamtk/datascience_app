# 非心分布

通常の t・カイ二乗・F は «帰無仮説が正しい» 前提の分布（中心分布）。
対立仮説が正しいとき、検定統計量は中心がずれた**非心分布**に従う。

- 非心度 $\lambda$ が «効果の大きさ»。$\lambda$ が大きいほど棄却域に入りやすく[検出力](power)が高い。
- 検出力やサンプルサイズ設計は、非心分布の «棄却域に入る確率» として計算する。
- 効果が 0 なら非心度 0 で中心分布に一致する。
