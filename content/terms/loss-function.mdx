# 損失関数

- **役割**: 予測 $\hat y$ と正解 $y$ のズレを 1 つの数 $L(\hat y, y)\ge 0$ にする。小さいほど良い予測。
- **回帰の二乗損失**: $L=(\hat y - y)^2$。外れ値に敏感。平均が[平均二乗誤差](mean-squared-error)。
- **分類の 0-1 損失**: 当たれば 0・外れれば 1。最適化しにくいので、交差エントロピーやヒンジ損失で代用する。
- 損失の «平均» が[経験誤差](empirical-risk)で、これを最小にするパラメータを選ぶのが学習。
- どの損失を選ぶかがモデルの «何を重視するか» を決める（外れ値への強さ、確率の較正など）。
