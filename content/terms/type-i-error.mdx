# 第一種の過誤（α エラー）

[帰無仮説](null-hypothesis) $H_0$ が**正しいのに棄却**してしまう誤り（偽陽性／「効果がないのに、ある」と言う誤り）。

- その確率は[有意水準](significance-level) $\alpha$ に等しい（$\alpha$ は許容上限として事前に決める）。
- 「あわてんぼうの誤り」とも呼ばれる。
- $\alpha$ を下げれば第一種の過誤は減るが、[第二種の過誤](type-ii-error) $\beta$ が増える（同じデータでは両方同時には下げられない）。
