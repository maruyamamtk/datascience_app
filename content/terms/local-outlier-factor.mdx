# 局所外れ値因子（LOF）

ある点の «局所的な密度» を、近傍点たちの局所的な密度の平均と比較することで **[外れ値](outlier)** を検出する手法。

1. **k距離**: ある点からk番目に近い点までの距離（**[k近傍法](k-nearest-neighbors)** と同じ考え方で、**[ユークリッド距離](euclidean-distance)** を使う）。
2. **到達可能距離**: reach-dist$(i,j) = \max(\text{k距離}(j), \text{dist}(i,j))$ ——近すぎる点どうしの距離を «底上げ» して平滑化する。
3. **局所到達可能密度（lrd）**: $\text{lrd}(i) = 1 / \text{（k近傍への到達可能距離の平均）}$ ——値が大きいほど密集している。
4. **LOF**: $\text{LOF}(i) = \dfrac{\text{近傍のlrdの平均}}{\text{lrd}(i)}$

- **LOF≈1**: 自分の密度が近傍と同程度——正常。
- **LOF≫1**: 自分の密度が近傍よりずっと «疎»——異常（局所的に浮いている）。
- 大域的な中心からの距離ではなく «近傍との相対比較» で異常度を測るため、密度が場所によって異なるデータ（クラスタが複数ある等）でも機能しやすい。
