# カーネル関数

[カーネル密度推定](kernel-density-estimation)で各データ点に置く «山» の形 $K(u)$。

- 条件: $\int K=1$、$K(u)=K(-u)$（対称）、平均0。
- 代表: **ガウス** $\frac{1}{\sqrt{2\pi}}e^{-u^2/2}$（無限の裾）、**エパネチニコフ** $\frac34(1-u^2)$（$|u|\le1$、理論上効率最良）、一様、三角。
- どのカーネルを使っても結果は大きくは変わらない。決定的なのは[帯域幅](bandwidth) $h$ の方。
- カーネルは «重み» の形。近い点ほど大きく寄与し、遠い点は小さく（または0）寄与する。
