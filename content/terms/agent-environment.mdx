# エージェントと環境

**[強化学習](reinforcement-learning)** は、次のループの繰り返しとして定式化される。

1. エージェントが現在の状態 $s$ を観測する。
2. エージェントが **[方策](policy)** に従って行動 $a$ を選ぶ。
3. 環境が行動を受け取り、次の状態 $s'$ と **[報酬](reward)** $r$ を返す。
4. 1に戻る。

- **エージェント**: 行動を決める主体（学習するのはこちら）。
- **環境**: エージェントの行動に応じて状態を遷移させ、報酬を生成する仕組み。多くの場合[マルコフ決定過程](markov-decision-process)として定式化され、環境の内部の遷移確率はエージェントには見えない（未知）ことが多い。
- グリッドワールドの例では、マス目上を動くキャラクターがエージェント、グリッドのルール（壁・ゴール・落とし穴の配置と、それに応じた報酬の出し方）が環境にあたる。
