# 混同行列

分類結果を «実際 × 予測» の度数表にまとめたもの。

|           | →1と予測 | →2と予測 |
| --------- | -------- | -------- |
| **実際1** | 正解     | 誤り     |
| **実際2** | 誤り     | 正解     |

- 対角が正しく分類、非対角が誤分類。[誤判別率](misclassification-rate)＝非対角の和/全体。
- 2クラスでは TP・FP・FN・TN に対応し、**感度**（実際1を1と当てる率）・**特異度**（実際2を2と当てる率）・適合率・F値が読める。
- クラスごとの «取りこぼし» の偏りが見えるので、単一の正解率より情報が多い。
